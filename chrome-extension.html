How to install the Yi Link chrome extension? (Beta)

Install the extension to add link easily.

1. Download the chrome extension.

Download

2. Unzip the "yil.ink-chrome-extension.zip" to your app folder.

3. Open the chrome extensions(chrome://extensions/).

4. Click "Developer mode" switch button to enable developer mode.

5. Click "Load unpacked" and selete the unzip folder to install extension.

How to use the Yi Link chrome extension? (Beta)

Use the chrome extension to set the image link address with one-click.

1. Hover the image anywhere and click the Yi Link button to copy image address.

2. Hover the image address input and click the Yi Link button to paste image address.
